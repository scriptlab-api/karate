# karate

This repository demonstrates a REST API automation framework built using the Karate framework.
The project focuses on validating real-world API workflows such as authentication, CRUD operations, and response validation.


## Project Objective

The goal of this project is to showcase API automation capabilities using a Behavior Driven Development (BDD) approach while maintaining a scalable and reusable test structure.


## API Workflows Covered

* Authentication and token generation
* Create resource
* Retrieve resource
* Update resource
* Partial update of resource
* Delete resource
* Response validation and status verification


## Tools & Technologies

* Karate Framework
* Java
* Maven
* REST APIs


## Project Structure

The framework is organized to maintain modular and maintainable API tests.

```
src
 └── test
     └── java
         ├── features
         ├── runners
         └── utilities
```


## Purpose of This Repository

This project is part of an API automation portfolio demonstrating how the same API workflows are automated using different frameworks and technologies by ScriptLab_API.
